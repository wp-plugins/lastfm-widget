== Description ==

Lastfm tracklist and album quilt.

== Installation ==

1. Upload `lastfm.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure it through the 'Appearance/Widgets' menu in WordPress
4. Not change color value, it's in test mode.

= What about lastfm-widget? =

It's easy to maintain clean your templetes and put where you want your lastfm tracklist or your album quilt.
